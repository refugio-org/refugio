refug.io Projekt
================

refug.io ist ein "Webshop" der Gutscheine zur Warenausgabe an Obdachlose
bereitstellt.  Das visuelle Userinterface unterstützt die Überwindung
sprachlicher Barieren.  Die Waren können an unterschiedlichen Orten in der
Stadt abgeholt werden.

Bei der Warenausgabe ist es notwendig die Bestellungen an die tatsächlichen
Mengen anzupassen.  Ah-hoc Bestellungen und eine Inventurfunktion bieten die
Möglichkeit die Warenbestände stetig zu aktualisieren.

Waren
-----

Die Waren sind als Warenpakete, einzelne Gegenstände oder größeneingeteilte
Waren nach Bedürfnissen organisiert.

* Badezimmer/Gesundheit
    * Zähneputzen
    * Waschen
    * Medizinische Hilfe
* Schlafzimmer/Camping
    * Slafsack (Kind, Normal, Groß)
    * Isomatte/Luftmatratze
    * Decke
    * Zelt
* Küche/Essen
    * Essensration
    * Brot
    * Süßigkeiten
    * Konservendose
* Kleidung
    * Jacke 
    * Hose
    * Unterwäsche
    * Schuhe
* OTHER
    * Legal Advice

Reservierung
------------
Eine Reservierung wird für einen bestimmten Ort getätigt. Die an diesem Ort
verfügbaren Waren werden angezeigt. Jeweils ein Artikel einer Gattung ist
selektierbar und wird sofort für eine kurze Zeit reserviert. Beim
Gutscheindruck wird die Dauer der Reservierung auf eine Zeit bis zur Abholung
verlängert.
